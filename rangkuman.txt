Rangkuman materi GIT week 1:
1. Intro : Dalam materi ini kita dikenalkan dengan tools yang akan digunakan dalam proses belajar pemrograman MERN. Beberapa tools tsb adalah Git, Github, Sourcetree, dan teks editor (VS Code, Sublime tex, Atom, dll). Selain itu kita juga disarankan untuk menginstal beberapa tools tambahan untuk mempermudah proses pembelajaran seperti Lightshot, Zoom, Screen O Matic, dan Grepper. Disini kita bisa mengetahui bahwa Git adalah VCS (Version Control System) yang berfungsi untuk merekam setiap perubahan yang terjadi pada dokumen/file/source code ketika kita melakukan editing.
2. Fundamental : Disini kita akan belajar bagaimana cara membuat repositori dan melakukan commit dengan menggunakan git bash atau terminal lainnya. selain itu proses clone, pull, dan push juga dipelajari dalam materi ini. Untuk mengetahui perubahan apa saja yang sudah dilakukan pada file, kita bisa melakukan perintah git log pada git bash lalu tekan enter, setelah itu akan muncul history perubahannya.
3. Branch : Materi ini membahas cara kita membuat branch dan merging melalui git di lokal komputer kita. Setelah kita bisa membuat repositori maka langkah selanjutnya adalah dengan membuat cabang dalam repo tersebut, salah satu manfaatnya adalah jika kita ingin membuat eksperimen fitur baru atau melakukan perubahan pada branch master tanpa takut merusak file originalnya. Kemudian ketika kita berhasil membuat fitur baru dan ternyata ingin diimplementasikan ke file originalnya maka kita bisa melakukan merging branch agar perubahan yang terjadi dapat diaplikasikan ke cabang master tempat file originalnya. Selain itu disini kita juga belajar mengatasi conflict apabila file original dan file dalam branch lain sama-sama dilakukan perubahan, salah satu cara untuk mengatasi conflict adalah dengan memilih perubahan mana yang ingin dipertahankan dan mana yang ingin dihapus.
4. Materi GUI : Dalam materi ini kita akan mempelajari Git versi GUI yaitu Sourcetree. Sourcetree adalah versi GUI dari Git, disini kita akan jauh lebih mudah memahami cara melakukan initialized, stagging, commit, clone, dsb. Hal ini dikarenakan kita tidak perlu mengetik comand teks seperti yang dilakukan pada saat menggunakan git.
