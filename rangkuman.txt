Rangkuman materi GIT week 1:
1. Intro : Dalam materi ini kita dikenalkan dengan tools yang akan digunakan dalam proses belajar pemrograman MERN. Beberapa tools tsb adalah Git, Github, Sourcetree, dan teks editor (VS Code, Sublime tex, Atom, dll). Selain itu kita juga disarankan untuk menginstal beberapa tools tambahan untuk mempermudah proses pembelajaran seperti Lightshot, Zoom, Screen O Matic, dan Grepper. Disini kita bisa mengetahui bahwa Git adalah VCS (Version Control System) yang berfungsi untuk merekam setiap perubahan yang terjadi pada dokumen/file/source code ketika kita melakukan editing.
